# AngadiHub

AngadiHub is a Full Stack E-commerce Web Application.

This project allows users to browse products, add items to cart, and manage orders.

## Technologies Used

Frontend:
- React
- HTML
- CSS
- JavaScript

Backend:
- Spring Boot
- Java

Database:
- MySQL

## Features

- User login and registration
- Product listing
- Add to cart
- Order management
- REST API integration

## Project Structure

AngadiHub
│
├── frontend  (React application)
└── backend   (Spring Boot application)

## How to Run the Project

1. Start the backend server using Spring Boot.
2. Run the React frontend using `npm start`.
3. Connect to MySQL database.
